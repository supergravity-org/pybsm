Welcome to pybsm's documentation!
=================================

pybsm implements the physical model called
**Basic Structures of Matter - Supergravitation Unified Theory**
as published by `Dr. Stoyan Sarg <http://www.helical-structures.org/>`_

The `Supergravity Foundation <https://supergravity.org>`_ develops this module further
investigate this model and develop novel application for the benefit of all humanity.
